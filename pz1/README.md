# Практичне заняття 1. Основи Python та робота з даними

1. Створіть програму, яка виводить на екран від 1 до 10.
2. Напишіть програму, яка знаходить середнє значення з трьох чисел, введених користувачем.
3. Реалізуйте програму, яка приймає на вхід рік народження користувача та виводить його вік.
4. Напишіть клас "Книга" з властивостями, такими як назва, автор та рік видання. Створіть об'єкт цього класу та виведіть його характеристики.
